Prado's Cleaning Website

Upload all files to a GitHub repository.

Files included:
- index.html
- about.html
- services.html
- gallery.html
- reviews.html
- contact.html
- style.css
- images folder

To replace the hero image:
1. Add your photo to the images folder.
2. Rename it hero-cleaning.jpg.
3. Upload it to GitHub.

To publish:
Repository > Settings > Pages > Deploy from branch > main > root > Save.
